База данных для сайта booking.com
База содержит данные по пользователям, отелям, местам их расположения, заведенным бронированиями,
доступным номерам в отелях и их характеристикам.
База призвана решать задачи:
- хранения и записи данных о доступных отелях
- хранения и записи данных о бронированиях
- хранения и записи данных о пользователях
- ведения учета забронированных номеров
- ведения учета бронирований пользователей
- передачи данных о бронировании и оплате номеров в системы бухгалтерского учета

Список таблиц:
users - пользователи
users_logins - пользовательские данные аккаунтовй
hotel_users - пользователи-владельцы отелей
hotels - отели
hotel_type - типы отелей
hotel_media - фотографии/видео отелей
hotel_rooms - номера отелей, с характеристикой.
hotel_rooms_media - фотов/видео номеров отелей
hotel_rooms_bookings - календарь бронирования номеров
media_types - типы медиа
booking_status_types - типы статусов бронировани
regions - география 
reviews - отзывы пользователей и оценки
messages - сообщения пользователей отелям
